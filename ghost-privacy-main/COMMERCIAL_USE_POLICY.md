This document outlines the commercial use policies for the Ghost project. The core software is open-source under AGPL v3.0, while certain aspects are protected for the project's sustainability.

## 3. Ghost Guardian (Enterprise Edition)
Ghost Guardian refers to the **enterprise-focused features and services** built on top of the core Ghost platform.

### Components:
- **Admin Dashboard**: User management, audit logs
- **Compliance Tooling**: HIPAA, GDPR assistance
- **Managed Deployment**: On-premise or private cloud
- **Support Services**: SLA, dedicated support
- **Integration APIs**: Custom enterprise integrations

### Licensing:
Ghost Guardian components require a **commercial license**. Revenue from Ghost Guardian licenses directly funds:
- Development of the core open-source Ghost project
- Security audits and research
- Infrastructure costs
- Community support
